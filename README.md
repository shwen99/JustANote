# JustANote

Windows 下的一个及简单的便笺小工具。

## WHY

包括 Windows 自带的便笺工具在内，已经有非常多的这类工具软件了，为什么我还要写一个？

* Windows 自带的便笺程序，会一直在任务栏显示便笺程序图标，这占用了宝贵的任务栏空间。

* 我希望便笺能够“嵌入”在桌面上，即使进入到显示桌面的状态，便笺也依然显示在那里。
  而 Windows 自带的便笺软件或相关工具都做不到这一点。

* 只是需要一个很简单便笺，能够随手记一段文字，不需要提醒/分类之类的附加功能

## 特色

这货最大的特色就是完全不占用系统资源，打开程序完成便笺的编辑之后，就可以关闭程序退出了，
他会将便笺内容完全的融合到桌面壁纸中。
